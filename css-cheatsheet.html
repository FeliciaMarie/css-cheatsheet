CSS Properties Cheat Sheet

Property Effect Example(s)
Font-family: Changes typeface of text font-family: Courier New; font-family: Helvetica;
Font-size: Adjusts size of text font-size: 25px; font-size: 20px;
Font-weight: Controls boldness of text font-weight: bold; font-weight: lighter;
Text-align: Tells how to align text text-align: center; text-align: right;
Color: Changes color of text or foreground color: blue color: maroon
Background-color Controls color of background background-color: teal; background-color: wheat;
Opacity: Changes transparency (set from 0-1) opacity: 0.5 opacity: 0.25
Background-image: Adds an image background image:
url("images/mountains.jpg");		 Brown Bear
List-style: Defines type of list list-style: lower-roman;
  1. One
  2. Two
  3. Three
 list-style: square;
  • Beans
  • Eggs
  • Tortillas
